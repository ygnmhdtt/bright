# bright

bright makes your display brighten or darken

## Table of Contents

* [Installation](#installation)
* [Examples](#examples)
* [License](#license)

## Installation

```
$ git clone git@github.com:ygnmhdtt/bright.git
$ cd bright
$ sudo cp bright /usr/local/bin/
```

## Examples

```
$ bright 1.0
$ bright 2
$ bright help
```
